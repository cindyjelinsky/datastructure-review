# Data Structures Study Repository

This repository was created for study purposes. It contains implementations of the most common data structures in code.

## Contents

- **Arrays** – basic operations and examples
- **Linked Lists** – singly and doubly linked list implementations
- **Stacks & Queues** – array- and list-based implementations
- **Hash Tables** – simple hash map implementations
- **Trees** – binary trees
